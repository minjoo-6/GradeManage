## 학생 학점 관리 프로그램

  
### 프로그램 소개
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
학생이 수강한 과목에 대한 학점을 입력 및 조회

  
### 주요 기능
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
- 학생 입력
- 교과목 입력
- 학생이 수강한 교과목에 대한 성적 입력 및 조회
